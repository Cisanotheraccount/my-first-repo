Digital Object 1: Spatial Canvases / P5.js 2D

This page follows the course tutorials for 2D Spatial Canvas P5, Managing Multiple P5.js Canvases, and 3D Canvas ThreeJS. I used separate canvas containers so each sketch has its own space on the webpage.

My design interest here is a soft computational composition: clean space, quiet color, and simple geometric systems that can shift from static layout into motion. The sketches connect my creative background in digital media, photography, animation, 3D modeling, and AIGC with a beginner-friendly coding workflow.

2D Static Drawing With Primitives

The first canvas is a static composition made with primitive shapes: rectangles, ellipses, lines, and triangles. It uses a light grid and layered forms to study balance, depth, and color.

Animated And Interactive Drawing

The second canvas is animated and interactive. Move the mouse across the canvas to bend the field; click inside the canvas to pause or restart the motion.

Digital Object 1: Spatial Canvases / Three.js 3D

The second part translates the same soft computational direction into 3D space. Each Three.js sketch creates a scene, camera, WebGL renderer, lights, primitive geometry, and an animation loop.

3D Primitives With Rotating Box

A simple scene with a rotating cube on a wireframe grid. This follows the beginner setup: scene, camera, renderer, lights, a primitive mesh, and an animation loop.

Interactive 3D Scene With Orbit Controls

An interactive scene with box, sphere, cylinder, cone, repeated small spheres, and orbit camera controls. Drag the canvas to orbit around the geometry.

Atmospheric Geometry

A fog-based scene with transparent primitives and layered depth. It keeps the tutorial's primitive geometry approach while adding a softer atmospheric mood.
