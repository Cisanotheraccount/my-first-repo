Digital Object 1: Spatial Canvases

This page follows the course tutorials for 2D Spatial Canvas P5 and Managing Multiple P5.js Canvases. I used p5.js instance mode so each canvas has its own sketch, its own variables, and its own container on the webpage.

My design interest here is a soft computational composition: clean space, quiet color, and simple geometric systems that can shift from static layout into motion. The sketches connect my creative background in digital media, photography, animation, 3D modeling, and AIGC with a beginner-friendly coding workflow.

2D Static Drawing With Primitives

The first canvas is a static composition made with primitive shapes: rectangles, ellipses, lines, and triangles. It uses a light grid and layered forms to study balance, depth, and color.

Animated And Interactive Drawing

The second canvas is animated and interactive. Move the mouse across the canvas to bend the field; click inside the canvas to pause or restart the motion.
